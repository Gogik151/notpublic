О игре Minecraft

Minecraft — это игра-песочница, созданная шведским программистом Маркусом Перссоном (Notch) и выпущенная компанией Mojang Studios в 2011 году. Игра позволяет игрокам строить конструкции из текстурированных кубов в 3D-мире.

Игра стала культурным феноменом и одной из самых продаваемых видеоигр всех времён, с более чем 300 миллионами проданных копий по всему миру.

Версии Minecraft

Java Edition

Оригинальная версия для ПК с полной поддержкой модов и самыми последними обновлениями

  • Поддержка модов
  • Команды и читы
  • Снапшоты и бета-версии
  • Кроссплатформенный мультиплеер

Bedrock Edition

Кроссплатформенная версия для мобильных устройств, консолей и Windows 10

  • Кроссплатформенная игра
  • Магазин Marketplace
  • Улучшенная производительность
  • Поддержка контроллеров

Education Edition

Специальная версия для образовательных учреждений с дополнительными инструментами

  • Образовательный контент
  • Инструменты для учителей
  • Безопасная среда
  • Готовые уроки
